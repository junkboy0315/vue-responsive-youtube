# What's this

This is Vue.js component to embed youtube videos to your project RESPONSIVELY.

# how to use

Copy `Youtube.vue` to your project and use it as follows:

```js
// maxWidth is optional
<Youtube videoId="8Hgt9HYaCDA" maxWidth="360px" />
```
